Об HTML, CSS и JavaScript

31 мая 2018 г.

JavaScript - лучший "друг" HTML и CSS. HTML задает разметку сайта, CSS отвечает за внешний вид, а JavaScript все это оживляет. С помощью кода на JavaScript программист определяет, как страница будет реагировать на действия пользователя.

На сегодняшний день JavaScript - единственный язык программирования для браузеров. Она работает практически на всех платформах Windows, MacOS, Linux, в том числе и на мобильных устройствах. Таким образом разработка интерактивных Web-приложений практически невозможна без знания JavaScript.

В 2009 году появился Node.js, который вывел JavaScript за пределы браузеров. Теперь его можно запустить практически на любом сервере.
