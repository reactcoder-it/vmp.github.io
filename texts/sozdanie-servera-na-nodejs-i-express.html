Создание простейшего сервера на Node.js и Express

17 августа 2018 г.

Express - это быстрый, гибкий, минималистичный веб-фреймворк для приложений Node.js. Посмотреть можно здесь.

Создание простейшего HTTP-сервера на базе Express начинается с установки самого Express с помощью следующей команды.

$ npm init
$ npm install --save express

Далее создаем главный файл с именем, например index.js и в нем пишем следующий код:

var express = require('express');
var app = express();

app.get('/', function(req, res) {
    res.send('Hello, world!');
});

app.listen(8080);

На этом все! Для запуска приложения в командной строке необходимо набрать следующую команду:

$ node index

Далее в браузере набираем путь 127.0.0.1:8080 и любуемся нашим приложением. Все достаточно просто!
